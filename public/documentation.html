GET /movies/:title

Description: Returns a movie by title.

Method: GET

URL: /movies/:title

üë§ User Endpoints

GET /users

Description: Returns all users.

Method: GET

URL: /users

Response:

[
  {
    "id": 1,
    "name": "John",
    "favoriteMovies": []
  },
  ...
]

POST /users

Description: Creates a new user.

Method: POST

Request Body Example:

{
  "name": "Alice"
}

Response: Created user object with ID

PUT /users/:id

Description: Updates an existing user's name.

Method: PUT

Query Parameters: id

Request Body Example:

{
  "name": "New Name"
}

Response: Updated user object

PUT /users/:id/:movieTitle

Description: Adds a movie to a user's favoriteMovies.

Method: PUT

Query Parameters: id, movieTitle

Response: Confirmation message

DELETE /users/:id/:movieTitle

Description: Removes a movie from a user's favoriteMovies.

Method: DELETE

Query Parameters: id, movieTitle

Response: Confirmation message

DELETE /users/:id

Description: Deletes a user by ID.

Method: DELETE

Query Parameters: id

Response: Confirmation message
